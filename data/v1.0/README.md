## ⚠️ Archived Development Versions

The `v1.0/` folder contains archived development builds kept for educational and historical reference.

### Do Not Use

The following versions are considered obsolete and are not supported:

* v1.0
* v1.1
* v1.2
* v1.3

These builds may contain bugs, unfinished functionality, and outdated architecture.

### Use Instead

* ✅ v1.4 (minimum recommended legacy version)
* ✅ Latest release (recommended)
* ✅ Current `main` branch (for contributors)

Contributors should avoid basing new work on versions earlier than v1.4.
